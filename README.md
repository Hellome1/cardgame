# 三国卡牌

一个基于 Electron + React + TypeScript 开发的桌面端卡牌游戏，类似三国杀。

## 项目结构

```
card-game/
├── electron/              # Electron 主进程
│   ├── main.ts           # 主进程入口（TypeScript源码）
│   ├── main.js           # 编译后的主进程
│   └── main.cjs          # CommonJS格式主进程
├── src/
│   ├── types/
│   │   └── game.ts       # 游戏类型定义
│   ├── game/
│   │   ├── CardManager.ts      # 卡牌管理
│   │   ├── CharacterManager.ts # 武将管理
│   │   ├── GameEngine.ts       # 游戏引擎
│   │   ├── AIPlayer.ts         # AI逻辑
│   │   └── DistanceCalculator.ts # 距离计算
│   ├── store/
│   │   └── gameStore.ts        # 游戏状态管理（Zustand）
│   ├── components/
│   │   ├── Card/               # 卡牌组件
│   │   ├── PlayerAvatar/       # 玩家头像组件
│   │   └── GameBoard/          # 游戏主界面
│   ├── utils/
│   │   └── Logger.ts           # 日志系统
│   ├── App.tsx
│   └── main.tsx
├── package.json
├── vite.config.ts
└── tsconfig.json
```

## 技术栈

- **Electron** - 桌面应用框架
- **React** - UI框架
- **TypeScript** - 类型安全
- **Zustand** - 状态管理
- **Vite** - 构建工具

## 安装和运行

### 安装依赖

```bash
cd card-game
npm install
```

### 开发模式

```bash
# 启动 Vite 开发服务器
npm run dev

# 启动 Electron（新终端）
# macOS/Linux
npm run electron:dev

# Windows
npm run electron:dev:win
```

### 构建发布版本

```bash
# Windows
npm run dist

# 输出目录: release/
```

## 游戏规则

### 身份分配

- **4人局**: 1主公、1忠臣、1反贼、1内奸
- **5人局**: 1主公、1忠臣、2反贼、1内奸

### 胜利条件

- **主公/忠臣**: 消灭所有反贼和内奸
- **反贼**: 消灭主公
- **内奸**: 成为最后存活的角色

### 回合流程

1. **回合开始阶段** - 触发回合开始时的技能
2. **判定阶段** - 处理延时锦囊
3. **摸牌阶段** - 从牌堆摸2张牌
4. **出牌阶段** - 使用卡牌和技能
5. **弃牌阶段** - 弃置超出体力值的手牌
6. **回合结束阶段** - 触发回合结束时的技能

### 攻击范围与距离

- **基础距离**: 座位之间的最短距离
- **-1马**: 减少你与其他角色的距离
- **+1马**: 增加其他角色与你的距离
- **武器**: 提供攻击范围（1-5）

## 卡牌类型

### 基本牌

| 卡牌 | 效果 |
|------|------|
| 杀 | 对攻击范围内的角色造成1点伤害 |
| 雷杀 | 造成1点雷电伤害（黑桃/梅花）|
| 火杀 | 造成1点火焰伤害（红桃/方块）|
| 闪 | 抵消【杀】的效果 |
| 桃 | 回复1点体力 |

### 锦囊牌

| 卡牌 | 效果 |
|------|------|
| 决斗 | 与一名角色拼【杀】，由目标先出，双方轮流出【杀】，无法出【杀】的一方受到1点伤害 |
| 火攻 | 对目标造成1点火焰伤害 |
| 顺手牵羊 | 获得距离1角色的一张牌 |
| 过河拆桥 | 弃置一名角色的一张牌 |
| 桃园结义 | 所有角色回复1点体力 |
| 万箭齐发 | 所有其他角色需出【闪】 |
| 南蛮入侵 | 所有其他角色需出【杀】 |
| 无中生有 | 摸两张牌 |
| 无懈可击 | 抵消锦囊牌效果 |

### 装备牌

#### 武器

| 武器 | 范围 | 效果 |
|------|------|------|
| 诸葛连弩 | 1 | 使用【杀】无次数限制 |
| 青釭剑 | 2 | 无视目标防具 |
| 雌雄双股剑 | 2 | 对异性角色有特殊效果 |
| 青龙偃月刀 | 3 | 杀被闪后可继续出杀 |
| 丈八蛇矛 | 3 | 可将两张手牌当【杀】使用 |
| 贯石斧 | 3 | 杀被闪后可弃牌强制命中 |
| 方天画戟 | 4 | 最后一张手牌杀可额外指定目标 |
| 麒麟弓 | 5 | 造成伤害可弃置目标坐骑 |

#### 防具

| 防具 | 效果 |
|------|------|
| 八卦阵 | 需要闪时可判定，红色视为打出闪 |
| 仁王盾 | 黑色【杀】无效 |
| 藤甲 | 普通杀、南蛮入侵、万箭齐发无效；火焰伤害+1 |
| 白银狮子 | 防止多余伤害；失去时回复1点体力 |

#### 坐骑

| 坐骑 | 效果 |
|------|------|
| +1马（绝影/的卢）| 其他角色计算与你的距离+1 |
