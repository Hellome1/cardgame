# 三国卡牌

一个基于 Electron + React + TypeScript 开发的桌面端卡牌游戏，类似三国杀。

## 项目结构

```
card-game/
├── electron/              # Electron 主进程
│   └── main.ts           # 主进程入口
├── src/
│   ├── types/
│   │   └── game.ts       # 游戏类型定义
│   ├── game/
│   │   ├── CardManager.ts     # 卡牌管理
│   │   ├── CharacterManager.ts # 武将管理
│   │   └── GameEngine.ts      # 游戏引擎
│   ├── store/
│   │   └── gameStore.ts       # 游戏状态管理
│   ├── components/
│   │   ├── Card/              # 卡牌组件
│   │   ├── PlayerAvatar/      # 玩家头像组件
│   │   └── GameBoard/         # 游戏主界面
│   ├── App.tsx
│   └── main.tsx
├── package.json
├── vite.config.ts
└── tsconfig.json
```

## 技术栈

- **Electron** - 桌面应用框架
- **React** - UI框架
- **TypeScript** - 类型安全
- **Zustand** - 状态管理
- **Vite** - 构建工具


### 开发模式

```bash

# 启动 Electron（新终端）

```

### 构建发布版本

```bash

```

## 游戏规则

### 身份分配

- **4人局**: 1主公、1忠臣、1反贼、1内奸
- **5人局**: 1主公、1忠臣、2反贼、1内奸

### 胜利条件

- **主公/忠臣**: 消灭所有反贼和内奸
- **反贼**: 消灭主公
- **内奸**: 成为最后存活的角色

### 回合流程

1. **回合开始阶段** - 触发回合开始时的技能
2. **判定阶段** - 处理延时锦囊
3. **摸牌阶段** - 从牌堆摸2张牌
4. **出牌阶段** - 使用卡牌和技能
5. **弃牌阶段** - 弃置超出体力值的手牌
6. **回合结束阶段** - 触发回合结束时的技能

## 卡牌类型

### 基本牌

- **杀**: 对攻击范围内的角色造成伤害
- **闪**: 抵消【杀】的效果
- **桃**: 回复1点体力

### 锦囊牌

- **决斗**: 与一名角色拼【杀】
- **火攻**: 展示手牌并造成伤害
- **顺手牵羊**: 获得距离1角色的一张牌
- **过河拆桥**: 弃置一名角色的一张牌
- **桃园结义**: 所有角色回复1点体力
- **万箭齐发**: 所有其他角色需出【闪】
- **南蛮入侵**: 所有其他角色需出【杀】
- **无中生有**: 摸两张牌

### 装备牌

- **武器**: 增加攻击范围
- **防具**: 提供防御效果
- **+1马**: 增加与其他角色的距离
- **-1马**: 减少与其他角色的距离
